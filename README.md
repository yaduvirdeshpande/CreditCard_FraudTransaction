# CreditCard_FraudTransaction

## Description

This project is based on Machine Learning for detecting fraudulent credit card transactions.  
The system analyzes transaction data and predicts whether a transaction is genuine or fraudulent.

The dataset contains anonymized transaction features (`V1` to `V28`), transaction amount, time, and class labels.

### Objectives
- Detect fraud transactions
- Perform data preprocessing
- Train machine learning models
- Evaluate model performance

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

- In this project, the following models were used for fraud detection:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- XGBoost Classifier (optional but gives better accuracy)
- Support Vector Machine (SVM)

## Best Model

After evaluation, the best performing model is selected based on:
- Precision
- Recall
- F1 Score
- Confusion Matrix
